# Immersive Input — AI 输入增强工具

基于 [pot-app/pot-desktop](https://github.com/pot-app/pot-desktop) 构建的 Windows AI 输入增强工具，提供文本润色、翻译、解析、对话等功能，所有操作均可通过鼠标划词或快捷键触发。

---

## 快速上手

### 首次运行配置

1. 启动程序后，右键任务栏托盘图标 → **偏好设置**
2. 进入左侧菜单 **AI 功能**，填写：
   - **API URL**：如 `https://api.openai.com/v1/chat/completions`（兼容任何 OpenAI 格式的接口）
   - **API Key**：你的 API 密钥（`sk-...`）
   - **模型**：如 `gpt-4o-mini`、`deepseek-chat`、`Qwen/Qwen2.5-7B-Instruct`
3. 点击 **测试连接** 验证配置
4. 保存设置

## 功能一览

### ⚡ 轻AI润色（核心功能）

将文字交给 AI 进行润色改写，同时生成 3 种不同风格的版本供选择。

#### 触发方式

| 方式 | 操作 |
|------|------|
| **# 符号触发** | 在任意输入框中输入文字，末尾加 `#`，**Ctrl+C 复制**，窗口自动弹出（需在设置中开启） |
| **划词工具栏** | 鼠标拖选文字 → 浮动工具栏出现 → 点击 ⚡ **轻AI** |
| **全局快捷键** | 在设置 → 快捷键中配置，如 `Alt+S`，触发后对当前选中文字操作 |
| **托盘菜单** | 右键托盘图标 → **轻AI润色** |

#### 窗口功能

- **3 种润色风格**并发生成：严谨审慎 / 结构清晰 / 口语自然
- **快捷模板**：缩写、扩写、纠错、改正式、改口语、翻译英文（一键填入附加要求）
- **附加要求**：底部输入框，可输入具体指令（如"去掉感叹号"），回车重新生成
- **✉ 精炼**：填写附加要求后，可对某个版本单独精炼，不影响其他版本
- **应用**：点击后将结果自动粘贴回原输入框，并保存历史
- **复制**：仅复制到剪贴板，不关闭窗口
- **拖动**：拖动标题栏可移动窗口位置
- **关闭**：点击 ✕ 关闭，或按 Esc

#### 个性化

在 **偏好设置 → AI 功能 → 个人写作偏好** 中填写你的风格偏好（如"我偏好简短直接，不用感叹号"），AI 会自动参考这段文字润色。

---

### 🌐 翻译

划词后弹出翻译结果，支持多种翻译服务。

#### 触发方式

| 方式 | 操作 |
|------|------|
| **划词工具栏** | 拖选文字 → 点击 🌐 **翻译** |
| **直接划词** | 偏好设置中关闭"划词后先显示工具栏"，拖选后直接翻译 |
| **输入翻译** | 托盘图标 → **输入翻译**，或快捷键触发 |
| **全局快捷键** | 设置 → 快捷键 → 划词翻译 |

---

### ❓ 解析

对选中文字进行详细解释，支持在同一窗口追问。

#### 触发方式

| 方式 | 操作 |
|------|------|
| **划词工具栏** | 拖选文字 → 点击 ❓ **解析** |
| **托盘菜单** | 右键托盘（暂不在菜单中，通过工具栏触发） |

#### 窗口功能

- AI 自动解释选中内容（含义、用法、背景知识）
- 底部输入框可追问，支持多轮对话
- **复制**按钮复制全部解析内容

---

### 💬 AI 对话

自由的多轮对话窗口，不限主题。

#### 触发方式

| 方式 | 操作 |
|------|------|
| **托盘菜单** | 右键托盘图标 → **AI 对话** |
| **配置窗口内** | 暂不直接暴露快捷键，可通过托盘触发 |

#### 窗口功能

- **气泡式对话 UI**，AI 回复渲染为 Markdown（支持代码块、列表、加粗等）
- **清空对话**按钮重置会话
- **停止**按钮终止生成
- 支持 Shift+Enter 换行，Enter 发送
- **拖动**：拖动标题栏移动位置
- 对话历史自动保存到本地数据库

---

### ✨ 文本格式化

一键规范化文本格式（不调用 AI，本地执行）。

#### 触发方式

- 划词工具栏 → 点击 ✨ **格式化**

#### 功能

- 中英文之间自动补空格（`你好World` → `你好 World`）
- 标点符号自动转换（中文后用全角，英文后用半角）
- 常见缩写规范化（api → API、ui → UI、id → ID 等）
- 多余空格和空行清理

---

### 🖼️ 划词浮动工具栏

选中文字后，鼠标附近自动出现小工具栏。

| 按钮 | 功能 |
|------|------|
| 🌐 翻译 | 打开翻译弹窗 |
| ❓ 解析 | 打开解析弹窗 |
| ✨ 格式化 | 就地格式化并粘贴回去 |
| ⚡ 轻AI | 打开轻AI润色窗口 |

- **自动隐藏**：无操作 6 秒后消失
- **触发模式**：可在偏好设置中选择"自动弹出"或切换回直接翻译模式

## 设置说明

在 **偏好设置（双击托盘图标）** 中配置：

### AI 功能页

| 设置项 | 说明 |
|--------|------|
| API URL | OpenAI 兼容接口地址 |
| API Key | 服务商密钥 |
| 模型 | 模型名称，如 gpt-4o-mini |
| 温度 | 0-2，越高越随机 |
| 个人写作偏好 | 附加到所有轻AI请求的个性化描述 |
| # 触发轻AI | 复制以 # 结尾的文本时自动触发 |
| 划词后先显示工具栏 | 关闭则划词直接翻译 |
| 自定义风格 Prompt | 分别编辑三种润色风格的系统提示词 |

### 快捷键页

| 快捷键 | 默认 | 功能 |
|--------|------|------|
| 划词翻译 | 未设置 | 划词后触发翻译 |
| 输入翻译 | 未设置 | 打开输入翻译窗口 |
| 轻AI快捷键 | 未设置 | 直接触发轻AI |

### AI 历史页

查看并管理轻AI / 解析 / 对话的历史记录：
- 按功能分类展示最近 100 条记录
- **导出 Markdown**：保存历史到文件
- **清空**：按类型清空
- **生成分析报告**：AI 分析你的表达习惯，给出改进建议

## 技术栈

- **前端**：React + Vite + Tailwind CSS + NextUI
- **后端**：Rust + Tauri v1
- **AI 调用**：直接调用 OpenAI 兼容 API，流式输出
- **历史存储**：SQLite（via tauri-plugin-sql）
- **配置存储**：JSON（via tauri-plugin-store）

## 常见问题

**Q: 轻AI弹出窗口后没有内容？**  
A: 检查偏好设置 → AI 功能中的 API 配置是否填写完整，点测试连接确认。

**Q: # 触发不工作？**  
A: 需要在设置中开启"# 符号触发轻AI"开关，然后复制以 # 结尾的文字（Ctrl+C）。

**Q: 划词后工具栏没出现？**  
A: 确认设置中"划词后先显示工具栏"已开启，且选中的文字超过 1 个字符。

**Q: 粘贴回去的文字不对？**  
A: 轻AI窗口点"应用"后，程序会尝试恢复焦点到原窗口并模拟 Ctrl+V。如果失败，可用"复制"按钮手动粘贴。
